Conectar Google Sheets
Configura tu Google Cloud Client ID para sincronizar con tu Google Sheets personal. Solo lo haces una vez.
1
Ir a Google Cloud Console
Abre console.cloud.google.com → crea un proyecto nuevo (ej: "FinanzApp").
2
Habilitar Google Sheets API
En tu proyecto ve a APIs y servicios → Biblioteca → busca Google Sheets API → Habilitar.
Haz lo mismo con Google Drive API.
3
Crear credenciales OAuth
Ve a APIs → Credenciales → Crear credencial → ID de cliente OAuth 2.0.
Tipo: Aplicación web.
En "Orígenes autorizados" agrega la URL donde usas la app (ej: https://tuusuario.github.io o http://localhost).
4
Pegar el Client ID abajo
Copia el Client ID generado (termina en .apps.googleusercontent.com) y pégalo aquí.
Omitir por ahora (usar solo localStorage)
